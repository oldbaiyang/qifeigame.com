# 极限脑力：动态数人头

一款类似任天堂《脑锻炼》的 H5 记忆力训练游戏。

## 游戏玩法

1. **观察初始人数** - 屏幕中央显示初始人物
2. **房屋降落** - 房子从天而降，遮挡人物
3. **观察进出** - 人物从左侧进入房屋，或从房屋走出到右侧
4. **输入答案** - 计算房屋内最终剩余人数
5. **挑战升级** - 正确答案进入下一关，难度递增

## 技术栈

- **Vue 3** - Composition API + TypeScript
- **Vite** - 快速构建工具
- **PixiJS** - 高性能 2D 渲染引擎
- **GSAP** - 流畅动画库
- **Howler.js** - 音频管理

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

- ✅ 生命值系统

- ✅ 移动端优化
- ✅ 对象池优化性能

## 游戏规则

- 初始 3 条生命
- 答对进入下一关，获得分数
- 答错扣除 1 条生命，重新挑战当前关卡
- 生命值耗尽游戏结束
